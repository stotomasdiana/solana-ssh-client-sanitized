# Privacy Policy

Last updated: April 2, 2026

## Overview

SSH Client is designed to provide mobile SSH access and wallet-assisted transaction flows. This repository is a public source-code distribution and does not itself operate a hosted backend service.

## Information We May Process

Depending on how the app is configured and used, the app may process:

- Server connection details entered by the user
- Authentication material such as passwords or private keys provided by the user
- Wallet addresses and transaction signatures returned by a compatible wallet
- Terminal input and output generated during active SSH sessions
- Basic device-level app data required for local storage and app functionality

## How Information Is Used

Information may be used solely to:

- Save server profiles locally on the device
- Establish SSH connections requested by the user
- Support wallet connection and transaction signing workflows
- Display terminal output and transaction status inside the app
- Improve reliability, debugging, and app maintenance in self-hosted or developer-managed builds

## Local Storage

The app may store certain user-provided credentials and configuration data locally on the device. Public repository exports intentionally exclude local signing files, local SDK configuration, and other machine-specific materials.

## Third-Party Services

The app may interact with third-party systems chosen by the user, including:

- SSH servers
- Solana RPC endpoints
- Solana-compatible wallets through Solana Mobile Wallet Adapter

Each third-party service is governed by its own privacy practices and terms.

## Data Sharing

This repository does not provide a hosted data-sharing service. Any transmission of data occurs only as part of direct app functionality initiated by the user, such as SSH communication or blockchain transaction submission.

## Security

Reasonable efforts may be made to store sensitive information securely on-device. However, no software or storage method can guarantee absolute security. Users are responsible for validating their server configuration, wallet actions, and deployment environment.

## Children's Privacy

This project is not directed to children under 13, and it is not intended for use by anyone who cannot legally consent to the applicable terms in their jurisdiction.

## Changes

This Privacy Policy may be updated from time to time. Continued use of the app after changes are published constitutes acceptance of the revised policy.

## Contact

For ownership, licensing, or privacy questions, use the contact details maintained by the repository owner or deployment operator.
